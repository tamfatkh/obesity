# obesity

This is part of the working Python code and the publicly available data used in the obesity paper titled "To be disclosed" by Dr. Tamer Oraby, Ms. Wei Yin, and Dr. Leyla Feize. We use non-spatial vs spatial ML and DL methods to predict country-level obesity.

Put all of the files in the same folder, which I call "obesity." Use the Python Jupyter notebook to run the code. Make sure to use the correct path to the folder "obesity" at the beginning of the code. The following is where you want to put your path to the folder "obesity."

################################

import os

os.makedirs(your_path, exist_ok=True)
